## What is Vibrant?


### Ignore Patterns


Vibrant supports custom rules via configuration file:

```javascript

  ignore: ["node_modules", ".git"],
  rules: {
    "generic-comment": "warn",
    "no-explicit-any": "error",
  },
};
```
